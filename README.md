# raspberrypi-radio

This project aims at giving a web layer to the piradio project:
http://www.dronkert.net/rpi/radio.html


Install

```shell
sudo ln -s ~/raspberrypi-webradio/bin/piradio /usr/local/bin/piradio
sudo chmod 755 /usr/local/bin/piradio
```
